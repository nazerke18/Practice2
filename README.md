# Practice 2 – Python Control Flow Basics

## Description
This repository contains Python code examples for learning
Python control flow structures.

The examples are based on the W3Schools Python tutorial.

## Topics Covered
- Boolean values
- Comparison operators
- Boolean operators (and, or, not)
- If statement
- If else statement
- If elif else statement
- Short hand if else
- While loop
- Break and continue
- For loop

## Project Structure
Practice2/
- boolean/   – Boolean examples
- if_else/   – If / else examples
- loops/     – While and for loop examples
